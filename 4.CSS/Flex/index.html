Display:Flex

Display: flex is a CSS property that makes an element a flex container. This means its direct child elements become flex items, which are arranged in a flexible way — either in a row or a column.

Display:Flex, Flex Wrap

  • This is a combination of two CSS properties used together to create a flex container that allows its child elements (flex items) to wrap onto multiple lines when necessary.
  • The three values of flex-wrap are:
    • No Wrap
    • Wrap
    • Wrap Reverse

Display:Flex, Flex Wrap: no wrap

  • All items stay on a single line, even if they overflow the container.

Display:Flex, Flex-direction

  • The flex-direction CSS property defines the direction in which flex items are placed in the flex container.
  • It controls the main axis along which the items are laid out (either horizontally or vertically).
  • flex-direction values:
    1. row Default. Items are placed left to right in a horizontal line.
    2. row-reverse Items are placed right to left in a horizontal line.
    3. column Items are placed top to bottom in a vertical line.
    4. column-reverse Items are placed bottom to top in a vertical line.

Display:Flex, Flex-direction: Row

  • Items are placed horizontally from left to right (main axis is horizontal).

Display:Flex, Flex-Flow

  • flex-flow is a shorthand CSS property that combines: flex-direction & flex-wrap .
  • It defines both the direction of flex items and whether they should wrap onto multiple lines.
  • flex-flow property is a shorthand for:
    1. row wrap
    2. row nowrap
    3. row wrap-reverse
    4. column wrap
    5. column nowrap
    6. column wrap-reverse

Display:Flex, Flex-Flow: Row wrap

  • Items go left to right. Wraps to the next line if needed (top to bottom).
